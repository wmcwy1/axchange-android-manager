## Axchange 是什么

Axchange 是一款专为 Mac 设计的安卓设备文件管理工具，底层用的是 ADB（Android Debug Bridge）技术，但把那套让人头疼的命令行包进了一个图形化界面里。

它的核心卖点其实就一句话：**手机不用装 App**。开启 USB 调试、连上数据线，Mac 这边就能像访问本地磁盘一样管理安卓设备里的文件。拖拽上传、拖拽下载、空格键快速预览图片音视频文档——这套交互对 Mac 用户来说几乎是肌肉记忆。

几个比较实在的功能点：

- **拖拽上传 / 下载**：批量操作，实时显示传输进度
- **空格键预览**：和访达（Finder）的 Quick Look 体验一致
- **地址栏跳转**：不用敲 `cd` 命令，直接编辑路径就行
- **文件重命名、新建文件夹、显示大小与权限**
- **快捷键支持**：全选、刷新、返回上级目录

需要提醒的是，拖拽下载和空格键预览这两个功能，仅支持 macOS 14.0 及更高版本。系统太老的话，部分体验会打折扣。
